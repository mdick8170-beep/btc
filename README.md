This project is a custom-built crypto wallet interface made entirely with HTML, CSS, and JavaScript. The wallet supports multiple cryptocurrencies like Bitcoin, Ethereum, Solana, XRP, USDT, and more, all inside a sleek modern UI inspired by real mobile crypto wallets. Users can switch between coins, view balances in both crypto and USD value, send or receive assets through interactive modals, and even generate simulated wallet addresses for different blockchains. All balances and transaction history are saved directly in the browser using localStorage, so everything stays persistent even after refreshing the page.

One of the hidden features built into the wallet is a quick-edit developer shortcut. By pressing the “F” key on the keyboard, users can instantly modify the balance of the currently selected coin without needing a backend or database. This makes the wallet easy to demo, test, and customize in real time. The app also includes transaction tracking, animated UI elements, glassmorphism styling, copy-to-clipboard functionality, and responsive design support for both desktop and mobile devices. Although it’s currently a frontend simulation and not connected to a real blockchain, the project demonstrates how a realistic crypto wallet experience can be recreated entirely in the browser.
